ELI'S SNAKE

Score: 5

Instructions:

    Get the red apples to score points, but beware of the green apples, as they will take away points! You can go through the walls, but doing so will take 2 points away. The goal of the game is to get 30 points.

  • Use arrow keys to move the snake
  • Space: Pause/Resume
  • Enter: Start game
  • Backspace: Reset game
  • Escape: Exit game

    • HAVE FUN!
